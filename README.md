 BMI & Calorie Calculator

A modern, interactive **BMI and Daily Calorie Calculator** built with **pure HTML, CSS and JavaScript**.
Includes a real-time **3D body character**, WHO-standard BMI classification, calorie goal modes, ideal weight range, and full macro breakdown.

---

## Features

* Real-time BMI calculation (WHO standards)
* 3D body character that morphs based on BMI and height
* Drag to rotate the 3D character
* Ideal weight range with visual comparison bar
* Calorie goal modes: Lose Weight / Maintain / Gain Muscle
* Macro breakdown — Protein, Carbs, Fats
* Animated calorie arc gauge
* Weekly weight change estimate
* Male / Female body types
* Fully responsive design
* No install needed — just open the HTML file

---

## Technologies Used

* **HTML5**
* **CSS3** — animations, grid, custom properties
* **Vanilla JavaScript** — all calculations, DOM updates
* **Three.js r128** — 3D character rendering

---

## Calculations

* **BMI** — `weight (kg) / height (m)²`
* **BMR** — Mifflin-St Jeor Formula (male & female)
* **Daily Calories** — `BMR × Activity Factor`
* **Ideal Weight** — WHO healthy BMI range (18.5–24.9)
* **Macros** — Protein / Fats as % of total calories, Carbs from remainder
